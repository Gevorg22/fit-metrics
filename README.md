# fitMetrics

**[Открыть приложение](https://fit-metrics-xi.vercel.app)**

Минималистичный дневник тренировок. Логинишься по одноразовому коду на почту, выбираешь упражнение из базы 1323 упражнений с GIF-анимациями, записываешь подходы с весом и повторениями. История, прогресс, личные рекорды - всё в одном месте.

## Возможности

### Авторизация и доступ
* **OTP по почте** - вводишь почту, получаешь 6-значный код, вводишь его. Без паролей. Email регистронезависим.
* **Гостевой режим** - можно попробовать приложение без регистрации, тренировки не сохраняются.
* **PWA** - устанавливается на домашний экран телефона как нативное приложение.

### Тренировки
* **1323 упражнения с GIF-анимациями** из ExerciseGymGifsDB, все названия переведены на русский.
* **Поиск** по названию и группе мышц (Грудь, Спина, Ноги, Пресс, Плечи, Бицепс, Трицепс, Кардио).
* **Избранные упражнения** - жми на сердечко рядом с упражнением, чтобы добавить в избранное. Отдельная вкладка "Избранное" в фильтре категорий. Для авторизованных пользователей избранное хранится в БД, для гостей в localStorage.
* **Таймер тренировки** - реальное время с момента старта.
* **Таймер отдыха** - кольцевой индикатор с обратным отсчётом и звуковым сигналом (60/90/120/180 сек).
* **Шаблоны тренировок** - сохрани набор упражнений и повтори одним нажатием.
* **Заметки** к тренировке.
* **Защита от пустой тренировки** - нельзя сохранить без упражнений.

### История и аналитика
* **История тренировок** - список с датой, длительностью, подходами. Фильтр по упражнению. Курсорная пагинация (кнопка «Загрузить ещё»).
* **График прогресса** - нажми на упражнение в Личных рекордах и увидишь как менялись вес и повторения.
* **Личные рекорды** - лучший вес по каждому упражнению + расчётный максимум на 1 повторение по формуле Эпли.
* **Карта мышц** - SVG силуэт тела с тепловой картой по нагрузке за последние 7 дней.
* **Календарь активности** - GitHub-style heatmap: видно регулярность тренировок за год.
* **Поделиться тренировкой** - генерирует карточку итогов.
* **Экспорт в CSV** - забрать всю историю тренировок.

### Вес
* **Дневник веса** - ввод веса тела за день.
* **График** динамики веса с фильтром по периоду.
* Удаление записей.

### Профиль
* Суммарный поднятый вес за всё время.
* Среднее время тренировки.
* Топ упражнений по количеству подходов.
* Количество тренировок и дней активности.

### Интерфейс
* **Светлая и тёмная тема** - переключение в шапке, сохраняется между сессиями, нет вспышки при загрузке.
* Адаптивный дизайн для мобильных и десктоп.

## Локальный запуск

### 1. Клонируй репозиторий

```bash
git clone https://github.com/Gevorg22/fit-metrics.git
cd fit-metrics
```

### 2. Установи зависимости

```bash
npm install
```

### 3. Настрой переменные окружения

Создай `.env.local` и заполни:

```env
DATABASE_URL="postgresql://user:password@localhost:5432/fitmetrics"
DIRECT_URL="postgresql://user:password@localhost:5432/fitmetrics"
AUTH_SECRET="your-secret-key"
NEXTAUTH_URL="http://localhost:3000"
RESEND_API_KEY="re_xxxxxxxxxx"
EMAIL_FROM="fitMetrics <noreply@yourdomain.com>"
```

### 4. Примени схему БД и засей упражнения

```bash
npm run db:push
npx tsx prisma/seed-gifdb.ts
```

### 5. Запусти сервер разработки

```bash
npm run dev
```

Открой [http://localhost:3000](http://localhost:3000).

## Скрипты

| Команда | Описание |
|---|---|
| `npm run dev` | Запуск в режиме разработки |
| `npm run build` | Сборка для продакшена |
| `npm run start` | Запуск собранного приложения |
| `npm run db:push` | Применить схему Prisma к БД |
| `npm run db:migrate` | Создать и применить миграцию |
| `npm run db:studio` | Открыть Prisma Studio |
| `npm run lint` | Проверка кода |
| `npx tsx prisma/seed-gifdb.ts` | Загрузить упражнения из ExerciseGymGifsDB |
| `npx tsx prisma/seed.ts` | Загрузить упражнения из старой базы (резервный) |
